1. **Initialize Git Repository**: Run `git init` in the project directory
2. **Configure Git User**: Set the user email to `minesunnya@gmail.com`
3. **Create .gitignore File**: Add entries for Python cache files, IDE-specific files, and large model files
4. **Stage Files**: Add all project files to git staging area
5. **Commit Changes**: Create an initial commit with descriptive message
6. **Add Remote Repository**: Set the remote URL to `https://github.com/CHANSingYeungSunny/photonic-edge-yolo`
7. **Push to GitHub**: Push the local branch to the remote repository
